## ğŸ”§ Hello World: Flask vs Express

### Express.js

```javascript
const express = require("express");
const app = express();

app.get("/", (req, res) => {
    res.send("Hello, World!");
});

app.listen(3000, () => {
    console.log("Server running on port 3000");
});
```

### Flask

```python
from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(port=3000)
```

### Key Differences

1. **`Flask(__name__)`** - Tells Flask where to find resources
2. **`@app.route()`** - Decorator instead of method call
3. **Return value** - Just return a string (or dict for JSON)
4. **No explicit `res` object** - Return value IS the response

## ğŸ¯ Why Flask for This Project?

We chose Flask because:

1. **Learning-friendly** - Simple enough to understand completely
2. **Production-ready** - Used by Pinterest, LinkedIn, Netflix
3. **Flexible** - No forced structure (like Django)
4. **Extensible** - Great ecosystem of extensions
5. **Similar to Express** - Easy transition from Node.js
6. **OpenAPI support** - flask-openapi3 for automatic docs
